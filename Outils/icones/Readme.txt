
These icon packs are intended to use on Windows 10 (1903+). Didn't tried on Windows 11


How to install: 
 
1. Delete "remove" after 7z from files names of 7TSP icon installers

2.Install Icon Pack with 7Tsp GUI.(https://www.deviantart.com/devillnside/art/7TSP-GUI-2019-Edition-804769422)
Run 7TSP GUI - click on Add a Custom Pack - browse to 7TSP...7z file - click on Start Patching - wait patiently for few minutes until asked to reboot. To remove icon pack,  on 7tsp GUI - click on Restore - reboot when asked.
If 7TSP GUI report some error, close it and open it again. Start patching again.

3. Because there are no windows live folders must use "disable_folder_thumbs.reg" - from folder "Disable icon thumbnails". Double click .reg file and restart.
Can always turn back, with "enable_folder_thumbs.reg" file.

At the end, for proper icons apperance, to refresh icon cache use "Refresh Icons.exe". But before that delete "remove" after file name


Comment installer :

Supprimez “remove” après 7z des noms de fichiers des installateurs d’icônes 7TSP.

Installez le pack d’icônes avec 7Tsp GUI. (https://www.deviantart.com/devillnside/art/7TSP-GUI-2019-Edition-804769422) Exécutez 7TSP GUI - cliquez sur Ajouter un pack personnalisé - parcourez le fichier 7TSP…7z - cliquez sur Démarrer le patch - attendez patiemment quelques minutes jusqu’à ce qu’on vous demande de redémarrer. Pour supprimer le pack d’icônes, sur 7tsp GUI - cliquez sur Restaurer - redémarrez lorsque vous êtes invité. Si 7TSP GUI signale une erreur, fermez-le et ouvrez-le à nouveau. Commencez à patcher à nouveau.

Parce qu’il n’y a pas de dossiers Windows Live, vous devez utiliser “disable_folder_thumbs.reg” - du dossier “Disable icon thumbnails”. Double-cliquez sur le fichier .reg et redémarrez. Vous pouvez toujours revenir en arrière, avec le fichier “enable_folder_thumbs.reg”.

Enfin, pour une apparence d’icônes appropriée, utilisez “Refresh Icons.exe” pour rafraîchir le cache des icônes. Mais avant cela, supprimez “remove” après le nom du fichier.
